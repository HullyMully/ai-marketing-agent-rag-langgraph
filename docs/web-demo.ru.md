# Веб-демо

[🇺🇸English](./web-demo.md) | 🇷🇺Русский

Приложение FastAPI отдаёт небольшой тёмный веб-интерфейс рядом с API. Все страницы
используют единую дизайн-систему, поэтому лендинг, чат, обзор API и метрики
выглядят как один внутренний продукт.

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env                  # по умолчанию офлайн-режим (без API-ключей)
python scripts/ingest_knowledge.py    # индексация базы знаний
uvicorn app.main:app --reload
```

## Страницы

- `http://localhost:8000/` — лендинг
- `http://localhost:8000/demo` — чат-демо
- `http://localhost:8000/api-overview` — оформленный обзор API (продуктовый вид эндпоинтов)
- `http://localhost:8000/metrics` — дашборд метрик (читает `/metrics/demo`)
- `http://localhost:8000/docs` — Swagger (без изменений)

## Чат-демо

В левой панели — быстрые действия. Каждое проигрывает чистый детерминированный
сценарий и заполняет правую панель **Workflow result**:

- **Lead creation flow** — квалифицирует потенциального клиента и показывает карточку лида
- **Services Q&A** — ответ из базы знаний
- **Pricing from RAG** — цены из базы знаний
- **Human escalation** — создаёт тикет и показывает карточку тикета
- **Memory follow-up** — ассистент повторно использует данные из этой же сессии

Быстрые действия показывают продуктового качества диалог и при этом тихо
обращаются к бэкенду, поэтому лиды, тикеты и метрики остаются настоящими. Если
ввести своё сообщение, выполняется живой `POST /chat` с реальным ответом.

## Рекомендуемые скриншоты

Снимайте в разрешении 1440×900:

1. **Лендинг** — `/`
2. **Создание лида** — запустите быстрое действие в `/demo`
3. **Эскалация** — запустите быстрое действие в `/demo`
4. **Обзор API** — `/api-overview`
5. **Дашборд метрик** — `/metrics`

Все демо-данные вымышленные и используют домен `.example`. Не оставляйте в кадре
токены и личные данные.
