# Кейс для портфолио — NovaGrowth AI Marketing Agent

[🇺🇸English](./portfolio-case-study.md) | 🇷🇺Русский

> **Это вымышленный бизнес-кейс и прототип для портфолио.** «NovaGrowth Agency» —
> не реальная компания, и эта работа **не** выполнялась для какого-либо реального
> клиента. Это *реалистичный демо-сценарий*, **вдохновлённый типичными рабочими
> процессами маркетинговых агентств**. Все данные синтетические.

## (Вымышленная) задача

NovaGrowth Agency — небольшое вымышленное агентство цифрового маркетинга —
получает много повторяющихся сообщений от потенциальных клиентов и сотрудников:
вопросы об услугах, ценах и процессе кампаний, а также реальные лиды и иногда
жалобы, требующие участия человека. Обрабатывать это вручную медленно и
непоследовательно.

**Цель:** сократить ручную работу с помощью разговорного ИИ-агента, который
отвечает из базы знаний, автоматически квалифицирует и заносит лиды в CRM и
эскалирует сложные случаи менеджеру-человеку, сохраняя контекст разговора.

## Прототип

Один сервис на FastAPI, предоставляющий агента на LangGraph, который:

- Отвечает на вопросы об **услугах / ценах / процессах** через **RAG** по
  синтетической базе знаний.
- **Квалифицирует лиды**: распознаёт намерение работать с агентством, собирает
  недостающие данные за несколько ходов и создаёт **лид в CRM**.
- **Эскалирует**: направляет явные запросы человека, жалобы или случаи с низкой
  уверенностью в **тикет поддержки** для менеджера-человека.
- **Помнит** данные в рамках сессии (имя, контакт, интересующую услугу).
- Доступен через REST и **Telegram-бот**.

## Почему эти технологии

| Требование | Реализация | Зачем |
|------------|------------|-------|
| Разговорная, состояниевая логика | конечный автомат **LangGraph** | Явный, прозрачный поток диалога |
| Промпты / абстракция LLM | **LangChain** | Стандартные блоки; лёгкая замена LLM |
| Ответы из документов | **RAG** + **Qdrant** | Обоснованные ответы, без выдуманных цен |
| CRM-подобные действия | инструменты + репозитории **SQLite** | Моделирует реальную интеграцию без SaaS |
| Эскалация | тикеты высокого приоритета | Чёткая передача человеку в цикле |
| Каналы | **FastAPI** + бот **aiogram** | Реалистичная мультиканальная поверхность |
| Воспроизводимость | **Docker Compose**, **pytest**, мок-режимы | Запускается где угодно, без ключей |

## Что демонстрирует (для роли Python AI Engineer)

Python, FastAPI, LangChain, LangGraph, RAG, векторную БД, разговорный ИИ,
состояниевую логику диалога, память, интеграции через API, CRM-подобные действия,
логику эскалации, Telegram-бот, Docker Compose, тесты и понятную документацию.

## Результаты (иллюстративные демо-метрики)

Эндпоинт `/metrics/demo` считает число разговоров, лидов, тикетов, долю эскалаций
и долю решённых ИИ по данным из SQLite — это те KPI, которые важны агентству.
Числа зависят от того, какие разговоры вы провели.
