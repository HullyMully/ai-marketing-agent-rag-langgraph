# Дорожная карта

[🇺🇸English](./roadmap.md) | 🇷🇺Русский

Возможные следующие шаги, примерно по приоритету. Это прототип для портфолио,
поэтому пункты — скорее направления, чем обязательства.

## Ближайшее
- Сделать настраиваемого провайдера LLM/эмбеддингов значением по умолчанию в профиле `prod`.
- **Извлечение полей лида через LLM** со структурированным выводом.
- **Потоковые** ответы чата (SSE / WebSocket).
- Хранить слоты памяти сессии в БД (переживать перезапуск, безопасно для нескольких процессов).

## Среднесрочное
- Реальный **коннектор CRM** (например, HubSpot/Pipedrive) за существующим слоем инструментов.
- **Аутентификация/авторизация** (API-ключи / OAuth) и ограничение частоты.
- **Стенд для оценки**: точность классификации интентов, доля попаданий поиска и
  проверки обоснованности ответов в CI.
- Более богатый **дашборд аналитики** (объёмы по интентам, воронка конверсии).

## Долгосрочное
- Поддержка нескольких языков (у бота уже есть EN/RU README).
- Интерфейс передачи диалога менеджерам для разбора эскалаций.
- Слой ограничителей / редактирования PII.
- Наблюдаемость: трассировка (LangSmith / OpenTelemetry) и экспорт метрик.
