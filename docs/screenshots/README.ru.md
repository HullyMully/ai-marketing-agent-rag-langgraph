# Скриншоты и визуальные материалы

[🇺🇸English](./README.md) | 🇷🇺Русский

Это демонстрационные материалы для кейса-портфолио. В них используются только
вымышленные данные.

## Материалы

Диаграммы (`docs/assets/`):

- `social-preview.svg` — обложка для социального превью репозитория (1280×640)
- `architecture-overview.svg` — архитектура системы
- `langgraph-flow.svg` — поток состояний агента
- `rag-pipeline.svg` — RAG-пайплайн

Скриншоты демо (`docs/screenshots/`):

- `landing-page.png` — лендинг
- `web-chat-lead-flow.png` — создание лида в чат-демо
- `api-overview.png` — страница обзора API
- `metrics-dashboard.png` — дашборд метрик

## Заметки

- Реальные данные клиентов не используются.
- Email используют домен `.example`.
- На скриншотах не должно быть токенов, API-ключей или приватных данных.

## Обновление скриншотов

Чтобы использовать снимки с запущенного экземпляра, поднимите приложение
(`uvicorn app.main:app --reload`), прогоните сценарии из
[../demo/demo-walkthrough.ru.md](../demo/demo-walkthrough.ru.md) и сохраните PNG в
эту папку. Затем укажите новые пути к изображениям в `README.md` и `README.ru.md`.
Снимайте в 1440×900 и не оставляйте в кадре токены и личные данные.
